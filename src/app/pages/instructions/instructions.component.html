You started a json server when running npm start. You can check the API available at localhost:4000.
Your first mission is to get the cupcakes from the API.
Add an instance of HttpClient in the api-service (src/app/shared/api-service) and fetch the localhost:4000/cupcakes endpoint.
In the CupcakeList component (src/app/pages/cupcake-list), use the service in order to assign the result of the http request to the attribute cupcakes.

Hint You have to create a class or an interface to describe and type a cupcake object (src/app/shared/cupcake.model.ts).

For this step, get the accessory list from the API.

Hint You will need an other attribute, accessories, to store the result. Don't forget to create your model...
